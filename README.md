## Installation

To run deploy dashboard

```bash
  npm install
  npm run dev
```

To run and deploy backend

```bash
  npm install
  npm run dev
```



## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`API_KEY`

`ANOTHER_API_KEY`

**Client:** React, Redux, TailwindCSS
